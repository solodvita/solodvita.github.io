Функция `alert()`
вызывает диалоговое окно.

Сумму двух переменных
можно вывести так:
``const a = 1
const b = 2
alert(a + b)``


Метод ```.toString()```
превращает числа в строку.

Его можно использовать так:

    const a = 1
    const b = 2

    (a + b).toString()
